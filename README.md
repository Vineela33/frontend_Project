# 🛒 FreshCart - Grocery Store Frontend Website

FreshCart is a responsive grocery store frontend website built using **HTML5** and **CSS3**. The project provides a modern and user-friendly interface for browsing grocery products, viewing product details, managing the shopping cart, and accessing company information.

## 📌 Features

- Responsive and clean user interface
- Home page with promotional banner
- Product listing page with categories and filters
- Product details page
- Shopping cart page
- About Us page
- Contact Us page
- Modern navigation bar and footer
- Mobile-friendly design
